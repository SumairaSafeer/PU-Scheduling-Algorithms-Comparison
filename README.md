## 📌 Project Overview

This project focuses on the **implementation and comparative analysis** of major **CPU Scheduling Algorithms** using C++. The goal is to simulate different scheduling techniques and evaluate their performance based on key metrics such as:

- Average Waiting Time
- Average Turnaround Time
- Average Response Time
- CPU Utilization
- Throughput

---

## 🛠️ Algorithms Implemented

| Algorithm              | Type                  | Description                              |
|------------------------|-----------------------|------------------------------------------|
| FCFS                   | Non-preemptive        | First Come First Serve                   |
| SJF                    | Non-preemptive / Preemptive | Shortest Job First                  |
| Round Robin (RR)       | Preemptive            | Time quantum based scheduling            |
| Priority Scheduling    | Preemptive / Non-preemptive | Priority-based scheduling         |
